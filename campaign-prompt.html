1）活动内容详情

活动概述

Prompt 共创计划面向所有社区用户收集可复用提示词。内容不只看一句提示词本身，还要能说明适用场景、推荐模型、输入变量和复用注意点；有结果图或视频案例时，其他用户更容易判断是否值得收藏和同款复用。

参与路径

1先投稿进入投稿编辑弹层，完成标题、提示词、适用场景和推荐模型。
2补案例同一条 Prompt 可以补充效果图或视频，让复用结果更可判断。
3被复用发布后的 Prompt 进入 AIGC 灵感广场详情弹层，其他用户可免费复制或一键同款。

奖励任务

1发布纯文字 Prompt写清使用场景、核心提示词、推荐模型和可复用注意点+20 积分 2补充图文/视频案例为同一条 Prompt 补充至少 1 张效果图或 1 段效果视频+40 积分 3免费复制/同款复用用户可在 Prompt 详情弹层复制正文，或一键同款带入轻创作免费

每日前 2 条有效投稿可计基础奖励；达到每日上限后仍可发布，只提示感谢共创，不再新增基础积分。复制正文和一键同款均为免费复用动作，同款会带入轻创作。

活动规则

先保证 Prompt 可读、可复用、可验证，再按内容完整度和推荐结果计算奖励。

  1. 发布入口：从活动详情页进入投稿编辑弹层，完成标题、提示词、适用场景和推荐模型后提交。
  2. 编辑要求：标题不少于 4 个字；中文正文不少于 30 字；英文或中英混合不少于 20 个英文单词；可选上传效果图或视频。
  3. 发布前校验：未达到标题、正文、场景、模型或案例要求时，发布按钮置灰，并在投稿编辑弹层直接提示缺失项。
  4. 发布成功反馈：纯文字 Prompt 发布成功获得 20 积分；带图文/视频案例发布成功获得 40 积分，并跳转到 AIGC 灵感广场自动打开 Prompt 详情弹层。
  5. 每日上限提示：达到每日奖励上限后仍可发布，但只提示“今日积分已达上限，感谢您的共创”。
  6. 免费复制复用：正文块内复制免费；一键同款免费带入轻创作，后续生成由轻创作页承接。
  7. 精选奖励：Prompt 被选入精选后，单条可再获得 100 积分；同一条 Prompt 只计一次精选奖励。
  8. 首页推荐奖励：Prompt 被推荐到首页后，单条可再获得 500 积分；同一条 Prompt 只计一次首页推荐奖励。
  9. 重复内容处理：高度相似、搬运、符号堆砌或无意义内容会被拦截或下架，已获得积分可追回。
