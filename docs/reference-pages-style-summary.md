# 五个基准页面的风格与组件总结

基准页面：首页、AIGC、模型广场、闪念、官方教程。本文只提炼现状，不要求修改这五个页面。

## 1. 总体风格

- **暗场沉浸**：近黑底色承接大图、视频和内容，背景层级主要靠透明度、轻边界与局部光晕区分。
- **电影感媒体**：首屏和核心卡片优先使用大幅媒体，文字叠在暗部或渐隐遮罩上，减少传统后台式大面板。
- **暖金主色**：暖金用于主操作、选中态、价格与积分；酸橙色只用于活动等高识别入口；白色用于中性操作。
- **低噪音玻璃**：导航、Tab、按钮和浮层使用低透明玻璃面，不采用高亮实心色铺满。
- **克制动效**：Hover 只做轻微上移、媒体缩放和边界增亮；进入动效保持短距离、低幅度。
- **编辑式排版**：大标题紧凑，正文使用低对比浅色；内容通过留白和媒体比例建立层级。

## 2. 导航组件

- 固定在视口顶部，高度 `76px`，背景为近透明黑色渐变并带轻模糊。
- 左侧使用完整品牌 Logo。
- 主导航固定为：首页、AIGC、模型广场、闪念、官方教程。
- 当前主导航显示线型图标，并使用深色胶囊承载；其他项保持纯文字。
- 右侧固定为：活动中心、AI 商城、通知、积分与头像组合胶囊。
- 活动中心使用酸橙色识别；AI 商城使用中性白；通知使用圆形图标按钮；积分与头像不拆成两个孤立控件。

## 3. 主要 UI 组件

### 按钮

- 主按钮：暖金文字与低透明暖金底，胶囊圆角。
- 次按钮：暗色玻璃底、轻边界、白色文字。
- Hover：上移 `1–2px`，边界和光晕增强，不改变组件尺寸。

### Tab 与筛选

- 外层为深色玻璃胶囊，内部间距 `4px`。
- 选中项使用暖金文字、暖金浅底和轻描边。
- 未选项保持低对比文字，Hover 只增加浅白背景。

### 卡片

- 以媒体、标题、元信息和动作构成，圆角约 `16–18px`。
- 边界对比低，阴影轻；避免“一个大框再套多层小框”。
- 媒体 Hover 轻微放大，正文位置不跳动。

### 表单与选择器

- 高度约 `48–52px`，暗色玻璃底与轻描边。
- Focus 使用暖金描边和窄外光环。
- Placeholder 明度低于正文，选中值保持正常字重。

### 弹层

- 深色强玻璃面，圆角约 `20px`。
- 结构按“媒体预览 / 说明与动作”组织。
- 关闭、下载、收藏、分享等操作统一使用线型图标。

## 4. 次级页面应用原则

- 不修改五个基准页面，所有统一样式通过 `secondary-pages.css` 仅加载到其余页面。
- 次级页面导航结构与首页完全一致，不增加“邀请有礼”等额外顶级入口。
- 活动、商城、通知、积分及详情页保留原有信息架构和功能，只调整视觉层级与共享组件。
- 顶部宣传横幅保留在 DOM 中但默认隐藏，避免把统一导航向下推移。
- 页面背景、卡片、Tab、按钮、表单和弹层复用同一组暗场、暖金、胶囊和轻玻璃规则。
